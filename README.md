# Agentic RAG System

A Retrieval-Augmented Generation (RAG) system built using LangChain, OpenAI Embeddings, and ChromaDB.

## Features

- Web document ingestion
- Vector embeddings
- Chroma vector database
- Semantic retrieval
- Document relevance grading
- Hallucination detection
- Answer validation

## Installation

```bash
pip install -r requirements.txt
```

## Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_api_key
```

## Run

```bash
python app.py
```
